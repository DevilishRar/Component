Frequently Asked Questions

Common questions about exploits and their status

What is UNC?

+

UNC (Unified Naming Convention) was an organization between executor developers to provide a unified scripting API for scripters. It helped resolve the issue of poorly named functions and created a standardized API that multiple executors could follow.

How often are the statuses updated?

Our status page automatically refreshes every minute to provide you with the most up-to-date information about exploit statuses and Roblox versions.

What's the difference between Script Executors and Externals?

Script Executors run scripts directly within the Roblox process, while External exploits operate outside the game process. Externals typically have different features and detection methods.

Why do some exploits show as "Detected"?

When an exploit is marked as "Detected", it means Roblox's security system has identified its signature. Using detected exploits may result in account warnings or bans.
